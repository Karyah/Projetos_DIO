(JDK)Java Development Kit significa Kit de Desenvolvimento Java, e é um conjunto de utilitários
 que permitem criar sistemas de software para a plataforma Java. 
 É composto por compilador e bibliotecas.

(JRE)Java Runtime Environment significa Ambiente de Tempo de Execução Java,
 e é utilizado para executar as aplicações da plataforma Java
